# 👨‍💻 its_me - Personal Portfolio

> Sebuah aplikasi web portofolio personal yang dibangun dengan Vue 3 dan Tailwind CSS untuk menampilkan profil, proyek, dan keahlian secara modern dan responsif.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Vue](https://img.shields.io/badge/Vue-3.5.13-brightgreen.svg)](https://vuejs.org/)
[![Vite](https://img.shields.io/badge/Vite-6.0.5-646cff.svg)](https://vitejs.dev/)

---

## ✨ Fitur Utama

- **🚀 Hero Section**: Perkenalan diri yang interaktif untuk menarik perhatian pengunjung.
- **👤 About Me**: Penjelasan detail mengenai latar belakang dan profil profesional.
- **💻 Projects Showcase**: Daftar proyek yang pernah dikerjakan lengkap dengan deskripsi.
- **📧 Contact Section**: Form atau area informasi untuk memudahkan orang lain menghubungi lo.
- **📱 Responsive Design**: Tampilan yang optimal di berbagai perangkat berkat Tailwind CSS.
- **⚡ Fast Performance**: Menggunakan Vite sebagai build tool untuk pengalaman pengembangan yang ngebut.

## 🛠 Teknologi yang Digunakan

Project ini dibangun dengan ekosistem modern:
- **Framework**: [Vue.js 3](https://vuejs.org/) (Composition API).
- **Styling**: [Tailwind CSS](https://tailwindcss.com/).
- **State Management**: [Pinia](https://pinia.vuejs.org/).
- **Build Tool**: [Vite](https://vitejs.dev/).
- **Linter/Formatter**: [ESLint](https://eslint.org/) & [Prettier](https://prettier.io/).

## 📋 Prasyarat Instalasi

Sebelum menjalankan project, pastikan lo sudah install:
- [Node.js](https://nodejs.org/) (Rekomendasi versi LTS terbaru).
- Package Manager: `npm` atau `yarn`.
